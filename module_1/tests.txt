Тестовый сценарий 1.1.1 Работа с корзиной
Предусловия:
Выбрать в "просмотре магазина" "все товары".
Добавить в корзину 2 элемента
...
Шаги:
- Шаг 1, Просмотреть корзину. Одидаемый результат - корзина отображается, в списке корзины доступны добавленные товары
- Шаг 2, Удалить товар из корзины. Ожидаемый результат - выбранный товар удален, список обновился, второй товар остался в списке
- Шаг 3, изменить количество товара в корзине. Кликнуть кнопку обновить. Ожидаемый результат - корректное изменение количества товара и корректный пересчет суммы в столбце "всего"
- Шаг 4, Ввести количество товара, превышающее число доступного на складе, ввести произвольное большое число, ожидаемый результат - невозможно, есть валидация поля
- Шаг 5, Клик на кнопку "У меня есть код скидки". О.Р. - появилось поле ввода, валидный код принят, значение "Всего" корректно пересчитано, не валиный код не прошел проверку, значение "всего" осталось прежнее.
